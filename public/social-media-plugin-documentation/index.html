Facebook Settings Guide
Requirements

To post to Facebook pages automatically, you need a Page Access Token and Page ID from a Facebook App.

Setup Steps
1. Create Facebook App
  • • Go to developers.facebook.com
  • • Click My Apps → Create App
  • • Choose "Business" as app type
  • • Enter App Name and Contact Email, then create
2. Generate Page Access Token
  • • Go to Tools → Graph API Explorer
  • • Select your app and choose your Facebook Page
  • • Add permissions: pages_manage_posts, pages_read_engagement, pages_show_list
  • • Click Generate Access Token and authorize
  • • Enter /me?fields=id,name and submit to get your Page ID
3. Configure Settings
  • • Paste the Page Access Token in the form
  • • Paste the Page ID in the corresponding field
Important Notes
  • Access tokens expire - use Graph API to extend them
  • Ensure permissions include pages_manage_posts, pages_read_engagement, pages_show_list
  • Apps in Development mode are restricted to admins/testers - submit for review for public access
Instagram Settings Guide
Requirements

To post to Instagram automatically, you need a Facebook Page Access Token (with Instagram permissions) and Instagram Business Account ID.

Setup Steps
1. Convert to Business Account
  • • Open Instagram app and go to your profile
  • • Tap the menu (three lines) and go to Settings and Privacy
  • • Account type and tools → Switch to Professional Account → Select Business
2. Link Instagram to Facebook Page
  • • Go to your Facebook Page (browser or app)
  • • Settings → Linked Accounts → Instagram
  • • Log in and connect your Instagram Business account
3. Generate Access Token
  • • Go to Tools → Graph API Explorer
  • • Select your app and choose your Facebook Page
  • • Add permissions: pages_manage_posts, pages_read_engagement, pages_show_list
  • • Click Generate Access Token and authorize
4. Get Instagram Business Account ID
  • • Using Page Access Token, query: /{facebook-page-id}?fields=instagram_business_account
  • • Copy the "id" from the instagram_business_account field
5. Configure Settings
  • • Paste Page Access Token in "Instagram Access Token" field
  • • Paste Instagram Business Account ID in corresponding field
Important Notes
  • Instagram account must be Business or Creator account linked to Facebook Page
  • Use long-lived Page Access Token to avoid frequent re-authentication
  • Ensure permissions include instagram_content_publish for posting
LinkedIn Settings Guide
Requirements

To post automatically to LinkedIn, you need an Access Token and either your Person ID (for personal profile) or Organization ID (for company page).

Setup Steps
1. Create a LinkedIn App
  • • Go to LinkedIn Developers
  • • Click "Create App", enter details, and link to a LinkedIn Page or Profile
  • • Complete the form and click "Create App"
2. Add Required Products
  • • In the App Dashboard, go to Products tab
  • • Request "Share on LinkedIn" and "Sign In With LinkedIn Using OpenID Connect"
3. Generate an Access Token
  • • Go to Docs and Tools → Auth Token Tools
  • • Select your app, scopes, and request Access Token
  • • Copy the Access Token from the response
4. Find Person or Organization ID
  • • Use your Access Token to query Person ID:
GET https://api.linkedin.com/v2/userinfo
Authorization: Bearer {ACCESS_TOKEN}
"sub" is the LinkedIn Person ID you will use for posting
  • • For Organization ID, query: https://api.linkedin.com/v2/organizationAcls?q=roleAssignee
  • • Copy the "id" or "sub" from the response
5. Add Token & ID in Settings
  • • Paste the Access Token into LinkedIn Access Token
  • • Paste the Person ID or Organization ID into LinkedIn Person/Org ID
Important Notes
  • Access Tokens usually expire in 60 days. Refresh when needed
  • Ensure permissions include w_member_social (profiles) or w_organization_social (pages)
  • If app is in development mode, only authorized users can post. Submit app for LinkedIn review for production use
Pinterest Settings Guide
Requirements

To integrate Pinterest, you need an App ID, App Secret, and App Type. Your account must be converted to a business account.

Setup Steps
1. Convert to Business Account
  • • Log in to Pinterest and go to your profile settings
  • • Under Account management, select "Convert to business account"
  • • Follow the conversion prompts
2. Create Pinterest App
  • • Visit developers.pinterest.com
  • • Click "Create App" and fill in required details
  • • Submit the form to create your app
3. Get App Credentials
  • • Go to your app details in the developer dashboard
  • • Copy the App ID and App Secret from credentials section
4. Determine App Type

New apps start as "Trial" with limited API access. Apply for "Standard" access in the Developer Portal for full functionality. Standard apps require Pinterest review and approval.

5. Configure Settings
  • • Enter your App ID in the Pinterest App Id field
  • • Enter your App Secret in the Pinterest App Secret field
  • • Select your App Type (Trial or Standard)
Important Notes
  • Business account is required for API access
  • Trial apps have limited API calls
  • Standard access requires Pinterest approval
