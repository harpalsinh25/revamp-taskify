Quick Start Guide

Get up and running with your first Taskify plugin in minutes.

1. Plugin Configuration

Create a plugin.json file to define your plugin:

JSON 
{
  "name": "Time Tracker",
  "slug": "time-tracker",
  "version": "1.0.0",
  "author": "Your Company",
  "description": "Track work, idle, break, and manual time logs for employees."
}

2. Service Provider

Bootstrap your plugin with a service provider:

PHP 
loadRoutesFrom(__DIR__ . '/src/routes.php');

        $this->loadMigrationsFrom(__DIR__ . '/database/migrations');
        $this->publishes([
            __DIR__ . '/config/timetracker.php' => config_path('timetracker.php'),
        ], 'config');
    }
}
?>

Plugin Structure

Follow this directory structure for maintainable plugins:

Directory Structure 
plugins/TimeTracker/
├── src/
│   ├── Controllers/
│   ├── Models/
│   ├── Services/
│   └── routes.php
├── resources/
│   ├── views/
│   └── lang/
├── database/
│   └── migrations/
├── config/
│   └── timetracker.php
├── PluginServiceProvider.php
└── plugin.json

Key Directories

  • Controllers - Handle HTTP requests and business logic
  • Models - Define data structures and relationships
  • Services - Business logic and reusable functions
  • Database - Migrations and seeders
  • Resources - Views and language files
  • Routes - Web and API route definitions
  • Config - Plugin configuration

Testing Your Plugin

Ensure reliability with Laravel's testing tools:

Feature Test

Test the Time Tracker controller:

create();
        $response = $this->actingAs($user)->postJson('/plugins/time-tracker/log', [

        $this->assertDatabaseHas('time_logs', [

Deploying Your Plugin

Deploy your plugin to production:

1. Package Your Plugin

Zip your plugin directory:

Bash 
cd plugins/TimeTracker
zip -r time-tracker.zip .

2. Install on Taskify

Install and migrate:

Bash 
php artisan plugin:install time-tracker.zip
php artisan migrate

3. Configure Environment

Ensure your environment is ready:

  • Update .env with database credentials.
  • Run php artisan config:cache.
  • Verify MariaDB is running in XAMPP.

4. Test in Production

Verify functionality:

Bash 
php artisan test --env=production
